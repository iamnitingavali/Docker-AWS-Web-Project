How the Stack Works

Each layer has a purpose. Here's how they connect.

🌍 Browser — User visits your domain
↓ HTTPS (port 443)
🌐 Nginx — Receives request, terminates SSL, routes traffic
↓ proxy_pass (port 3000)
⚙️ Node.js API — Business logic, auth, reads/writes DB
↓ TCP (port 5432)
🗄️ PostgreSQL — Stores users, sessions, data (Docker volume)
↑ All containers on Docker bridge network
